
# 🚀 Article-Web-Scraping: The Ultimate Sports News AI Assistant

## 📰 About The Project

Ever wished you could have a personal sports analyst who's read every single article published today? Well, that's exactly what this project delivers!

**The Evolution of a Simple Scraper:**
This project started as a basic Guardian API scraper, but it has evolved into something far more powerful. I've transformed it into an intelligent Q&A system that doesn't just fetch articles—it understands them!

Here's what makes this project special:
- 🔍 **Scrapes** the latest sports and football articles from The Guardian
- 🧩 **Breaks down** articles into intelligent chunks for better processing
- 🧠 **Stores** everything in FAISS (Facebook AI Similarity Search) for lightning-fast retrieval
- 💬 **Answers** your questions using state-of-the-art Hugging Face LLM models
- 🌐 **Showcases** everything through a sleek local Flask server with an intuitive HTML interface

The best part? You can ask questions like "What were the key highlights from yesterday's football matches?" and get instant, accurate answers drawn from dozens of articles!

## 🔄 Project Workflow

The project follows a clear three-step pipeline:

### Step 1: Fetch Data
```bash
python fetchData.py
```
- Fetches the latest sports and football articles from The Guardian API
- Saves raw article data with timestamps
- Stores articles in organized folders by date

### Step 2: Chunk and Build Index
```bash
python chunkData.py
```
- Breaks down long articles into meaningful, context-rich segments
- Creates embeddings using sentence transformers
- Builds a FAISS index for efficient similarity search
- Saves the index and metadata as a `.pkl` file

### Step 3: Run the Application
```bash
python app.py
```
- Loads the FAISS index and LLM model
- Starts the Flask local server
- Opens the HTML interface for user interaction
- Processes user queries and generates AI-powered responses

## 🚀 Getting Started

### Prerequisites
- Python 3.8 or higher
- Guardian API Key (free to obtain)
- 8GB+ RAM recommended for LLM models

### Installation from GitHub

1. **Clone the Repository**
   ```bash
   git clone https://github.com/mobina-tkd-85/Article-Web-Scraping-And-Searching.git
   cd Article-Web-Scraping
   ```

2. **Set Up Virtual Environment** (Recommended)
   ```bash
   conda create -p venv python=3.9 -y
   conda activate venv/
   ```

3. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Configure API Key**
   - Get your free API key from [The Guardian Developer Portal](https://open-platform.theguardian.com/access/)
   - Add it to the `config.py` file or set it as an environment variable

5. **Run the Complete Pipeline**
   ```bash
   # Step 1: Fetch articles
   python fetchData.py
   
   # Step 2: Chunk and build FAISS index
   python chunkData.py
   
   # Step 3: Launch the application
   python app.py
   ```

6. **Open Your Browser**
   - Navigate to `http://localhost:5000`
   - Start asking questions about the latest sports news!
